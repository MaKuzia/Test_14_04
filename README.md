## Создание и управление репозиторием
**git init** — инициализирует новый Git-репозиторий в текущей директории, создавая скрытую папку .git с необходимой структурой.


**git clone <repository_url>**— клонирует существующий удалённый репозиторий на локальный компьютер.


**git remote add <name> <url>** — связывает локальный репозиторий с удалённым. 

## Работа с изменениями и коммитами
**git add <file>** — добавляет файл для отслежтвания изменений (подготовку к коммиту).


**git --all** - добавляет все файлы в индекс.


**git add . **— добавляет все изменённые файлы.


**git commit** — создаёт коммит с подготовленными изменениями. Можно указать сообщение коммита с помощью опции -m.


## Работа с удалёнными репозиториями
**git push <remote-name> <branch-name>** — отправляет изменения из локального репозитория в удалённый.


**git pull** — получает изменения из удалённого репозитория в локальный и автоматически сливает их с текущей веткой.
